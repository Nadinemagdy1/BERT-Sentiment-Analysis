# BERT Sentiment Analysis Project

This project uses BERT (Bidirectional Encoder Representations from Transformers) for sentiment analysis.

## Objective
Classify text into positive or negative sentiment using a pre-trained BERT model.

## Tools & Libraries
- Python
- Transformers (HuggingFace)
- PyTorch / TensorFlow

## Group Members
- Nadine Magdy  
- Nour Mohamed

## How it works
The model reads text input and predicts sentiment using a pre-trained BERT pipeline.

## Output
The system classifies sentences into:
- Positive
- Negative
